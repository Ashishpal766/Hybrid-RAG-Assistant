## ✨ Key Features
* **Hybrid Intelligence:** Combines **BM25 (Keyword Retrieval)** and **MMR Vector Search** for unmatched accuracy.
* **Context-Aware:** Fully utilizes chat history to maintain conversational flow.
* **Strict PDF Fidelity:** Smart logic to prioritize PDF context and transparently flag external AI knowledge.
* **Enterprise-Grade LLM:** Powered by `Llama-3.3-70b` for deep reasoning and nuance.
* **Source Attribution:** Automatically provides source page references from your documents.
